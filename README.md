Installation
---
```bash
npm install itako-text-reader-simplayer --save
```

Development
---
Requirement global
* NodeJS v5.10.0
* Npm v3.8.3

```bash
git clone https://github.com/maxmellon/itako-text-reader-simplayer
cd itako-text-reader-simplayer
npm install

npm test
```
